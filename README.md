

# Gale-Shapely Algorithim Visualization

This is my visualization for the Gale-Shapely algorithim, created with React and P5. You can edit the amount of nodes, and set the order of preferences. Feel free to fork and add what you like! To learn and find out more about the Gale-Shapely algorithim, check out this wikipedia link.

https://en.wikipedia.org/wiki/Stable_marriage_problem
